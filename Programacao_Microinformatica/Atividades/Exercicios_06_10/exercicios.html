O que é DOM? (Clique para ver a resposta)

DOM significa Document Object Model. É uma interface de programação para documentos HTML, XML e SVG. Ele representa a página de tal forma que programas podem mudar a estrutura, o estilo e o conteúdo do documento. O JavaScript usa o DOM para interagir com os elementos da página.
